DEVELOPER TOOLS

  DISABLE CACHE
    - Network tab
    - "Disable cache" checkbox on the top

  SLOW DOWN NETWORK
    - Network tab
    - "No throttling" drop-down list on the top

  SAVE AND RUN SNIPPETS OF CODE
    - Go to Sources -> Snippets tab (may be hidden, click on >>)
    - To add a new script, press New snippet and name it, for example
      last-modification-date.js:
      console.log(new Date(document.lastModified));
    - Code and save it (Ctrl-S)
    - Press on play button to run (Ctrl-Enter)

  HTTP REQUEST HEADERS
    - Go to Network
    - Select XHR as filter
    - Optionally press right click on table headers and check on Method to show
      GET/POST in the table
